1-2주마다 사전 릴리스 버전이 최신 버전이 됩니다. 이때 잠기지 않은 모든 플레이어는 현재 버전으로 업그레이드됩니다. 어떤 이유로 특정 버전에서 플레이어를 잠그려면 다음을 참조하십시오. 자동 플레이어 업데이트 선택 해제(외부 사이트) .
 
 

새 버전 사용해보기

시험판 버전을 사용할 수있는 경우 시험판 버전을 사용해 보는 가장 쉬운 방법은 Studio의선수모듈에서 변경하려는 플레이어를 클릭 한 다음선수 정보 . 버전 선택기를 사용하여 최신 시험판을 선택하십시오.

curl 문을 사용하여 플레이어를 시험판 버전으로 업데이트 할 수 있습니다. 컬의 빠른 개요는 상단에서 찾을 수 있습니다. 단계별 : 플레이어 관리 (외부 사이트) . 문서에는 방법에 대한 특정 섹션이 있습니다. 플레이어 업데이트 .

다음은 플레이어를 업데이트하는 curl 명령입니다. 귀하의 이메일 , YOUR_ACCOUNT_ID과 YOUR_PLAYER_ID . 

curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL \
  --request PATCH \
  --data '{
    "player": {
    "template": {
      "version": "6.57.0"
      }
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/YOUR_ACCOUNT_ID/players/YOUR_PLAYER_ID/configuration

대부분의 경우 플레이어를 게시하고 싶을 것입니다. 다음 curl 문으로 수행 할 수 있습니다. 다시 말하지만, 표시된 곳에서 올바른 값으로 대체해야합니다. 

curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL  \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/YOUR_ACCOUNT/players/YOUR_PLAYER_ID/publish
