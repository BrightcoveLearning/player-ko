플레이어 모양 사용자 지정-레거시 스킨

이 문서에는 최신 스킨이 아닌 브라이트코브 플레이어 1.x 스킨 (그래파이트) 에 대한 정보가 포함되어 있습니다. 이 새로운 피부로의 전환은 2015년 12월 둘째 주에 일어났습니다. 문서의 최신 버전은이 문서를 참조하십시오 .

이 항목은 new / styling / customizing-player-appearance.html 플레이어의 모양을 사용자 지정하는 방법에 대한 입문서입니다. 콘텐츠는 Chrome의 개발 도구를 사용하지만 모든 최신 브라우저에는 동일한 기능이 있습니다. 여러 브라우저에서 개발 도구를 사용하는 방법에 대한 간략한 소개는자바 스크립트 디버깅 기초문서.

참고: 이 문서에서는 CSS에 대한 지식과 브라우저의 개발 도구를 사용하여 HTML 요소의 클래스를 확인하고 적용 가능한 CSS를 검토합니다. 필수 기본 지식이없는 경우 다음을 통해 작업하는 것이 가장 좋습니다. 빠른 시작: 플레이어 커스터마이즈 - 레거시 스킨 .

플레이어의 모양을 변경할 수 있지만, 먼저 플레이어 모양을 해결할 방법이 있어야 합니다. 이를 보려면 플레이어 embed_in_page 코드가 삽입 된 HTML 페이지를 찾아보세요. 에서집단개발 도구 섹션의 첫 번째<div>값이있는 클래스가있는 것을 볼 수 있습니다. video-js할당 됨.

이 사실을 알고 나면 스타일을 사용하여 플레이어 자체를 변경할 수 있습니다. 예를 들어 플레이어의 크기를 변경하려면 다음 스타일을 사용할 수 있습니다.

이프레임 플레이어

플레이어의 iframe 구현을 사용하는 경우 상황이 다릅니다. 여전히 플레이어가video-js클래스이지만 물론 iframe 내부에 있습니다.

만드는 스타일은 iframe에서 플레이어와 계속 작동하지만, CSS 파일을 만들고 Studio를 사용하여 플레이어와 연결해야 합니다. 당신은플레이어모듈을 클릭 한 다음 CSS를 연결하려는 동영상을 클릭 한 다음플러그인섹션에 CSS 파일 경로를 추가하십시오.

iframe 자체를 맞춤 설정하려면 다음을 추가하면됩니다. class속성iframe요소 (27 행). 이 경우 클래스 값은videoFrame . 그런 다음<style>태그를 사용하여 원하는 대로 수정합니다. 아래 예에서width과height변경됩니다.

      width: 640px;

    }

  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

재생 버튼

재생 버튼의 모양을 변경하려면 먼저 해결 방법을 알아야합니다. Chrome에서개발 도구열린 다음 버튼을 마우스 오른쪽 버튼으로 클릭하십시오. 표시되는 선택 항목에서요소를 점검하다 .

에서집단개발 도구 섹션에서 버튼 요소에 해당하는 HTML 코드를 볼 수 있습니다.

위의 강조 표시된 코드에서 버튼에 할당된 클래스가 인 것을 볼 수vjs-big-play-button있습니다. 재생 버튼의 색상 (호랑이와 일치) 과 불투명도는 다음과 같은 스타일로 변경됩니다.

  <style>
    .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

이 접근 방식은 효과가 있다고 생각하지만 그렇지 않습니다. CSS로 작업할 때는 속성 특수성을 알고 있어야 합니다. 이는 관련성에 따라 요소에 적용할 속성 값을 브라우저가 결정하는 방법을 나타냅니다.

버튼 변경 사항을 적용하는 한 가지 방법은 CSS 선택기에서 ID를 참조하는 것입니다. 이는 id의 CSS 특이성이 높기 때문입니다. 자세한 내용은 CSS 특이성부분

먼저id속성div플레이어 코드의 요소.

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    <script src="//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.min.js"></script>

그런 다음 CSS에서 플레이어의 큰 재생 버튼을 다음과 같이 참조합니다.

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

컨트롤 막대 아이콘 순서

컨트롤 막대에서 아이콘 순서를 제어 할 수 있습니다. order스타일에 번호가 지정되었습니다. 예를 들어 5진행률 표시 줄 뒤에서 볼륨 버튼 앞에 아이콘을 배치합니다. 9컨트롤 막대의 맨 오른쪽에 아이콘이 배치됩니다. 이전 브라우저의 경우 다음과 같이 더 많은 스타일이 필요할 수 있습니다.

{
  order: 5; /* after the progress bar but before the volume button */
  -webkit-box-ordinal-group: 5;
  -moz-box-ordinal-group: 5;
  -webkit-order: 5;
  -ms-flex-order: 5;
}

이 경우 파일을 다운로드하기 위해 아이콘을 추가하는 예가브라이트코브 플레이어 샘플에 나와 있습니다. 비디오 플러그인문서를 다운로드하십시오.

CSS 특이성

CSS 특수성은 브라우저가 관련성에 따라 요소에 적용되는 속성 값을 결정하는 방식을 말합니다. 이 개념에 대한 자세한 내용은특이성문서를 참조하십시오.

Chrome에서 큰 재생 버튼을 검사합니다. background속성은 여러 클래스를 사용하여 설정됩니다. 즉, 이 속성을 재정의하는 데 하나의.vjs-big-play-button클래스만 사용할 수 없습니다.

큰 재생 버튼을 커스터마이징하는 세 가지 방법은 다음과 같습니다.

  • 플레이어에 할당된 ID를 참조합니다.
  • 여러 클래스를 사용하여 속성의 관련성을 높입니다.
  • JavaScript를 사용하여 플레이어에서 id 속성을 설정한 다음 CSS에서 참조합니다.

id 참조

id속성은 CSS 특수성과 관련성이 높기 때문에 CSS 선택기에id를 추가하면background-color속성.

아직 하지 않았다면 추가id속성video다음과 같은 요소:

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

CSS에서 id 값을.vjs-big-play-button클래스 선택기에 추가합니다. 다음과 같이 배경색과 불투명도를 재정의합니다.

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

이제 큰 재생 버튼이 주황색임을 알 수 있습니다.

여러 클래스 사용

이후background큰 재생 버튼의 속성은 두 개의 클래스를 사용하여 설정됩니다.이 속성을 재정의하려면 CSS 선택기에 두 개 이상의 클래스가 필요합니다.

.video-js클래스 선택기에.vjs-big-play-button클래스를 추가합니다. .video-js클래스는 비디오 컨테이너에 설정되어 있음을 기억하십시오.

이 값을 사용하여 클래스 이름이 다음과 같은 모든 요소를 선택합니다. vjs-big-play-button클래스 이름이 다음과 같은 요소의 하위 요소입니다. .video-js .

<style>
  .video-js .vjs-big-play-button {
    background-color: #B37D5B;
    opacity: .6;
  }
  .video-js {
    width: 640px;

이제 큰 재생 버튼이 주황색임을 알 수 있습니다. 여러 클래스로 작업하는 것은 까다로울 수 있으며 여러 요소에 영향을 줄 수 있지만 id는 하나의 요소에 고유합니다.

큰 재생 버튼 요소에 ID를 추가하는 코드를 추가합니다.
