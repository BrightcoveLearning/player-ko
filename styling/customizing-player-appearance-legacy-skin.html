페이지 내용

    플레이어 모양 커스터마이징 - 레거시 스킨

    이 문서에는 최신 스킨이 아닌 브라이트코브 플레이어 1.x 스킨 (흑연) 에 대한 정보가 포함되어 있습니다. 새로운 피부로의 전환은 2015 년 12 월 두 번째 주에 일어났습니다. 최신 버전의 문서는이 문서를참조하십시오.

    이 항목은 new / styling / customizing-player-appearance.html 플레이어의 모양을 사용자 지정하는 방법에 대한 입문서입니다. 콘텐츠는 Chrome의 개발 도구를 사용하지만 모든 최신 브라우저에는 동등한 기능이 있습니다. 여러 브라우저에서 개발 도구를 사용하는 방법에 대한 간략한 소개는자바 스크립트 디버깅 기초문서.

    자신 만의 스킨 만들기

    기본 스킨을 사용하지 않고 자신만의 스킨을 생성하려는 경우 플레이어 관리 API를 사용하여 이 스킨을 생성할 수 있습니다. 플레이어 구성 가이드는필요한 정보를 제공합니다.

    플레이어

    참고: 이 문서는 CSS에 대한 지식과 브라우저의 개발 도구를 사용하여 HTML 요소의 클래스를보고 적용 가능한 CSS를 검사한다고 가정합니다. 필수 기본 지식이없는 경우 다음을 통해 작업하는 것이 가장 좋습니다. 빠른 시작: 플레이어 커스터마이징-레거시 스킨 .

    플레이어의 모양을 변경할 수 있지만 먼저 문제를 해결할 방법이 있어야 합니다. 이를 보려면 플레이어 embed_in_page 코드가 삽입 된 HTML 페이지를 찾아보세요. 에서집단개발 도구 섹션의 첫 번째<div>값이있는 클래스가있는 것을 볼 수 있습니다. video-js할당 됨.

    이 사실을 알고 나면 스타일을 사용하여 플레이어 자체를 변경할 수 있습니다. 예를 들어 플레이어의 크기를 변경하려면 다음 스타일을 사용할 수 있습니다.

    <style>
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

    iframe 플레이어

    플레이어의 iframe 구현을 사용하는 경우 상황이 다릅니다. 여전히 플레이어가video-js클래스이지만 물론 iframe 내부에 있습니다.

    만든 스타일은 iframe에서 플레이어와 계속 작동하지만 CSS 파일을 만들어 Studio를 사용하여 플레이어와 연결해야 합니다. 당신은플레이어모듈을 클릭 한 다음 CSS를 연결하려는 동영상을 클릭 한 다음플러그인섹션에 CSS 파일 경로를 추가하십시오.

    iframe 자체를 맞춤 설정하려면class속성iframe요소 (27 행). 이 경우 클래스 값은videoFrame . 그런 다음<style>태그를 사용하여 원하는대로 수정합니다. 아래 예에서width과height변경됩니다.

    <!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Customize iframe</title>

  <!-- Page styles -->
  <style>
    .videoFrame {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

    재생 버튼

    재생 버튼의 모양을 변경하려면 먼저 해결 방법을 알아야합니다. Chrome에서개발 도구열린 다음 버튼을 마우스 오른쪽 버튼으로 클릭하십시오. 표시되는 선택 항목에서요소를 점검하다 .

    에서집단개발 도구 섹션에서 버튼 요소에 해당하는 HTML 코드를 볼 수 있습니다.

    CSS 특이성

    CSS 특이성은 브라우저가 관련성에 따라 요소에 적용되는 속성 값을 결정하는 방식을 의미합니다. 이 개념에 대한 자세한 내용은특이성문서를 검토하십시오.

    Chrome에서 큰 재생 버튼을 검사합니다. background속성은 여러 클래스를 사용하여 설정됩니다. 즉, 이 속성을 재정의하기 위해 하나의.vjs-big-play-button클래스 만 사용할 수 없습니다.

    다음은 큰 재생 버튼을 사용자 지정하는 세 가지 접근 방식입니다.

    • 플레이어에게 할당 된 ID를 참조하십시오.
    • 여러 클래스를 사용하여 속성의 관련성을 높입니다.
    • 자바 스크립트를 사용하여 플레이어에서 id 속성을 설정 한 다음 CSS에서 참조 할 수 있습니다.
