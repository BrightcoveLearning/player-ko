스타일링 재생 목록

이 항목에서는 재생 목록 스타일을 지정하는 방법을 배웁니다.

개요

재생 목록의 모양을 변경하려면 CSS를 사용하면됩니다. 이 문서에서는 변경할 수있는 여러 스타일을 볼 수 있습니다. 물론 가능한 변경 사항은 많으며 작업을 수행하려면 올바른 선택기를 찾아야합니다.

다음 문서는 재생 목록 사용 그림을 완성하는 데 필수적인 정보입니다.
  • 재생 목록 구현 : 이 문서에서는 재생 목록 사용에 대해 어떤 결정을 내리는지, 실제로 HTML 페이지에서 플레이리스트를 가져오는 데 필요한 플레이어 코드를 얻는 방법을 보여줍니다.
  • 재생 목록 UI 플러그인 : 이 문서에서는 재생 목록 기능의 모양과 느낌을 향상시키는 데 사용할 수있는 옵션에 대해 자세히 설명합니다.

세로 재생 목록 스타일 지정

다음 다이어그램은 세로 재생 목록에 대해 변경된 스타일을 요약합니다. 다이어그램 다음은 스타일을 나열하는 표입니다.

재생 목록의 텍스트 색상 제어 .vjs-마우스.vjs-재생 목록 {
  색상: #E9D2B3;
}
의 색상 제어
재생 목록 항목 사이의 공백		 .vjs-마우스.vjs-재생 목록 {
  배경 색상: #161D6E;
}
지속 시간 텍스트의 색상 제어 .vjs-재생 목록.vjs-재생 목록-지속 시간 {
  색상: #9BB4DA;
}
지속 시간 배경의 색상 제어 .vjs-재생 목록.vjs-재생 목록-지속 시간 {
  배경 색상: #1160AA;
}
재생 목록 항목 사이의 공간 제어 .vjs-재생 목록-수직 .vjs-재생 목록-항목 {
  여백 상단: 20px;
}
첫 번째 재생 목록 항목 위의 여백을 방지하려면 다음과 같이 여백 설정 스타일을 따라야 합니다. 
.vjs-playlist-vertical .vjs-playlist-item:first-child {
  margin-top: 0px;
}

가로 재생 목록 스타일 지정

다음 다이어그램은 가로 재생 목록에 대해 변경된 스타일을 요약합니다. 다이어그램 다음은 스타일을 나열하는 표입니다.

재생 목록의 텍스트 색상 제어 .vjs-마우스.vjs-재생 목록 {
  색상: #E9D2B3;
}
의 색상 제어
재생 목록 항목 사이의 공백		 .vjs-마우스.vjs-재생 목록 {
  배경 색상: #161D6E;
}
지속 시간 텍스트의 색상 제어 .vjs-재생 목록.vjs-재생 목록-지속 시간 {
  색상: #9BB4DA;
}
지속 시간 배경의 색상 제어 .vjs-재생 목록.vjs-재생 목록-지속 시간 {
  배경 색상: #1160AA;
}
재생 목록 항목 사이의 공간 제어 .vjs-재생 목록-수평 .vjs-재생 목록-항목 {
  여백-왼쪽: 5픽셀;
}
첫 번째 재생 목록 항목의 왼쪽 여백을 방지하려면 다음과 같이 여백 설정 스타일을 따라야 합니다. 
.vjs-playlist-horizontal .vjs-playlist-item:first-child {
  margin-left: 0px;
}

스크롤바 숨기기

가로 재생 목록의 스크롤 막대는 필요하지 않지만 나타납니다. 스크롤바를 숨기려면 다음 스타일을 사용하십시오.

div.vjs-playlist.vjs-playlist-horizontal {
  overflow-y: hidden;
}

다음 스크린 샷은 재생 목록에 표시된 스크롤 막대를 보여주고 숨겨져 있습니다.
