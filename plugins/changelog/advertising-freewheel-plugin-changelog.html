    FreeWheel 플러그인 변경 로그를 사용한 광고

    다음은 플러그인에 대한 변경 사항을 추적합니다.

    변경로그

    분류: 3월 30일

    v3.5.2

    • 버그 수정: 허용하다play_request adstart 전

    2020 년 3 월 18 일

    v3.5.1

    • 다음에 대한 지원 추가minDuration과maxDuration temporalSlots에
    • 호환성 수정 Video.js 7.7.x

    3일 (12월)

    v3.5.0

    • 다음에 대한 지원 추가minDuration과maxDuration ...에temporalSlots

    2019 년 9 월 4 일

    v3.4.0

    • 광고 컨테이너의 크기를playerresize이벤트 발생
    • FW SDK URL 업데이트
    • VPAID 광고의 크기 문제 수정
    참고: 실제 FreeWheel 플러그인 문서에서 언급했듯이올바른 SDK URL 사용섹션에서 FreeWheel은 버전이 지정된 URL을 선호하는 동적 SDK URL을 없앴습니다. 이는 Brightcove Player 플러그인이 이제 기본적으로 최신 버전이 아닌 특정 버전으로 잠겨 있음을 의미합니다. Brightcove는 FreeWheel SDK 릴리스를 따라 잡으려고 시도하지만 뒤처 질 수 있습니다.

    브라이트코브 플레이어 통합에서 자신의 SDK 버전을 관리할 수 있습니다. 여기에는 FreeWheel 플러그인에Html5.sdkurl옵션을 전달하는 작업이 포함됩니다.

    player.FreeWheelPlugin({
  Html5: {
    sdkurl: 'preferred-sdk-url'
  }
});

    2019년 3월

    v3.3.1

    • 버그 수정: 제대로 설정하기 위해 FreeWheel SDK로 보낼 때 기간이 부동인지 확인하십시오. vdur URL의 매개 변수
    • 최신 코딩 표준 지원 (vjsstandard 8)

    2019 년 1 월 2 일

    v3.3.0

    • 자동 재생 처리 추가

    2018 년 8 월 13 일

    v3.2.0

    • Live Player 문서 PDF 링크 수정
    • 더하다isVpaid FreeWheel 플러그인에 대한 편리한 방법

    2018년 6월 13일

    v3.1.0

    • 광고 재생 중 플레이어 오버레이를 숨기는 동작 추가
    • 빈 VAST 응답으로 인해 Android에서 콘텐츠가 재생되지 않는 버그가 수정되었습니다.
    • VPAID 광고 클릭 후 콘텐츠를 재개 할 수없는 버그 수정
    • 플러그인 옵션을 통해 Freewheel SDK 매개 변수를 설정하는 기능 추가
    • 플러그인 옵션을 통해 Freewheel SDK 로그 수준을 설정하는 기능 추가

    2013월 7일

    v3.0.0

    • 기본 기술 주문이 HTML5 전용으로 변경되었습니다.
    • 기본값pauseOnAdClickthrough에서 변경되었습니다false ...에true
    • VPAID 클릭 연결 영역은 이제 플레이어와 함께 크기가 조정됩니다.
    • 추가됨loadWithCookie선택권
    • 소멸 된 제거nativeHlsForceHtml선택권
    • 플러그인은 이제 구성에서 상대 URL을 존중합니다.
    • 업데이트 됨videojs-contrib-ads 4.2.7에서 5.1.6으로
      • 플레이어가 처분 될 때 타임 아웃 지우기
      • 다음을 포함한보다 안정적인 미디어 이벤트loadeddata과loadedmetadata
      • 플러그인이 너무 늦게 초기화되면 오류를 내 보냅니다.
      • 특정 광고 오류로 인해 광고 모드가 종료되지 않는 버그를 수정했습니다.
      • Video.js 6과의 호환성 향상
      • iOS에서 광고 중에 자막이 표시 될 수있는 문제 해결
      • 고치다snapshot.trackChangeHandler곤충
      • 소스 변경 후 재생 상태 버그 수정
