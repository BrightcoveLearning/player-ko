Chromecast 플러그인 변경 로그

변경 내역

2020 년 9 월 8 일

v2.0.0

  • CAF API를 사용하여 다시 구현 된 Chromecast 수신기

21년 4월 20일

v1.3.2

  • 버그 수정: FreeWheel 광고가 재생되지 않으면 버튼이 표시되지 않음
  • 비정기적인 지속적인 로딩 상태 수정
  • 수신기 카탈로그 테스트 추가

v1.3.1

v1.3.0

  • 아랍어 번역 추가
  • 버그 수정: 최대 목표 버퍼 길이를 30으로 설정하여 추가 버퍼 오류 방지
  • 첫 글자가 아닌 전체 장치 이름 표시
  • 새로운 테스트 절차로 업데이트 된 README
  • 안전하지 않은 콘텐츠 테스트 단계로 README 업데이트
