개요

Custom Endscreen 플러그인을 사용하면 비디오 재생이 끝날 때 고객이 제공 한 HTML을 표시 할 수 있습니다.

아래 샘플 비디오는 Custom Endscreen 플러그인의 사용을 보여줍니다. 이 짧은 비디오의 끝에서 다음 HTML에서 표시되는 텍스트를 볼 수 있습니다.

<strong>Content</strong> for <em>custom</em> end screen

플레이어 모듈을 사용하여 구현

맞춤형 최종 화면은 다음과 같이 구현할 수 있습니다. 최종 화면 속성 편집또는 사용자 지정 최종 화면 플러그인을 사용합니다. 플레이어 모듈을 사용하여 사용자 정의 최종 화면 플러그인을 구현하려면 다음 단계를 따르십시오.

  1. 열기선수모듈을 열고 새 플레이어를 만들거나 플러그인을 추가하려는 플레이어를 찾습니다.
  2. 플레이어의 속성을 열려면 플레이어 링크를 클릭합니다.
  3. 왼쪽 탐색 메뉴에서최종 화면을 클릭합니다.
  4. 다음으로플레이어 종료 화면입력하고 아래쪽 화살표를 클릭하고커스텀 .
  5. 구성 옵션을 입력하십시오. 맞춤 HTML텍스트 상자. 
    {
  "content": "<strong>Content</strong> for <em>custom</em> end screen"
}
  6. 저장을 클릭합니다 .
  7. 플레이어를 게시하려면 [ 게시 및 포함] > [변경 사항 게시] 를 클릭합니다 .
  8. 열린 대화 상자를 닫으려면닫기를 클릭합니다 .

코드를 사용하여 구현

플러그인을 구현하려면 플레이어는 플러그인 코드의 위치, 필요한 경우 스타일시트, 플러그인 이름 및 플러그인 구성 옵션을 알아야 합니다. 플러그인 코드 및 스타일시트의 위치는 다음과 같습니다.

https://players.brightcove.net/videojs-custom-endscreen/2/videojs-custom-endscreen.min.js
https://players.brightcove.net/videojs-custom-endscreen/2/videojs-custom-endscreen.css

최종 화면 사용자 지정

CSS를 사용하여 최종 화면의 모양을 사용자 지정할 수 있습니다. 다음은 수행 할 수있는 작업의 몇 가지 예입니다. 구현을 테스트하여 페이지 콘텐츠에서 작동하는지 확인하세요.

이 섹션의 코드는 반응형 플레이어와 함께 사용할 수 있는 상대 위치 지정을 사용합니다. 플레이어를 반응형으로 만들려면반응형 크기 조정문서화하거나 사용선수설정하는 모듈사이징설정반응형 .

이것은 비디오 다시 시작 버튼과 소셜 공유 버튼이있는 최종 화면이 표시되는 방식입니다.

버튼 크기 변경

최종 화면 버튼의 크기를 변경하고 싶다고 가정 해 보겠습니다. 이러한 버튼 이미지는 SVG (Scalable Vector Graphics) 형식이므로 모든 플랫폼에서 빠르게로드됩니다. 이 때문에 크기를 변경할 수 없습니다. width과height CSS 속성.

대신 이러한 버튼은 XML 텍스트 파일이므로 다음을 사용하여 크기를 변경할 수 있습니다. font-size CSS 속성.

아래 코드를 사용하여 두 버튼을 더 작게 만드세요.

.vjs-overlay-buttons {
  font-size: .5em;
}

백분율 값을 사용하면 동일한 효과가 나타납니다.

.vjs-overlay-buttons {
  font-size: 50%;
}

또는 각 버튼의 크기를 독립적으로 제어 할 수 있습니다.

  font-size: .5em;
}
.vjs-trigger-social-control {
  font-size: .5em;
}

매크로 사용

그만큼content맞춤 최종 화면에 제공된 문자열에 이제 매크로가 포함될 수 있습니다 (videojs-contrib-ads에서 지원됨). 예를 들어 현재 플레이어에있는 동영상의 이름과 설명을 최종 화면에 포함하려면 다음과 유사한 코드를 사용합니다.

videojs.getPlayer('myPlayerID').on('loadstart', function() {
  var myPlayer = this;

  // +++ Call endscreen method +++
  myPlayer.customEndscreen({
    content: '{mediainfo.name} - {mediainfo.description}'
  });
});

그러면 다음과 같은 최종 화면이 나타납니다.

다음은 사용 가능한 매크로의 전체 목록입니다.

매크로 설명
{player.id} 플레이어 ID
{mediainfo.id} 비디오 ID
{mediainfo.name} 비디오 타이틀
{mediainfo.description} 간단한 설명 (최대 250자)
{mediainfo.tags} 비디오와 연결된 태그 (메타데이터)
{mediainfo.reference_id} 참조 ID
{mediainfo.duration} 비디오 클라우드에서 보고한 동영상의 지속 시간
{mediainfo.ad_keys} Studio의 Media 모듈에서 추가 및 편집할 수 있는 자유 형식 텍스트 문자열; 양식에서 query 매개 변수를 사용해야 합니다. 
  cust_params={mediainfo.ad_keys}
{player.duration} 플레이어가 측정한 비디오 지속 시간 (아마도 약간 다를 수mediainfo.duration있으며 더 정확할 수 있음)
{document.referrer} 참조 페이지 URL
{window.location.href} 현재 페이지 URL
{timestamp} 1/1/70 이후의 현재 현지 시간 (밀리초)
{random} 난수 0-1 조 (독특한 인상을 만드는 데 사용됩니다. 이렇게 하면 광고가 브라우저에 캐시되지 않고 노출 불일치를 방지할 수 있습니다.)
