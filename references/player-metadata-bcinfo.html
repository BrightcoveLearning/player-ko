bcinfo의 플레이어 메타 데이터

이 항목에서는 플레이어에 대한 정보 (메타 데이터)를 검색하는 방법을bcinfo목적.

개체 개요

다음은 특정 사례입니다. console.log()그리고console.table()에 포함 된 데이터보기bcinfo특성.

다음과 유사한 코드를 사용하여이 정보에 액세스 할 수 있습니다.

videojs.getPlayer('myPlayerID').ready(function() {
  var myPlayer = this;
  console.log('myPlayer.bcinfo', myPlayer.bcinfo);
  console.table(myPlayer.bcinfo);
});
NS bcinfo개체가 다음에 채워집니다. videojs()호출됩니다(또는 플레이어 스크립트가 실행됨). 따라서 객체의 정보를ready()방법은 바로 위의 코드에서 볼 수 있듯이 안전한 방법입니다.

개체 속성

그만큼bcinfo속성은 현재 플레이어에 대한 정보 (메타 데이터)를 포함하는 객체입니다. 개체에있는 속성은 다음과 같습니다.

필드 이름 설명 데이터 유형
계정 ID Brightcove 계정 식별자 끈
응용 프로그램 ID 분석과 함께 사용하기 위해 설정할 수있는 값입니다. 플레이어의 속성으로 정의됩니다.
