개발 시퀀스

권장되는 개발 시퀀스는 다음과 같습니다.

  1. 인페이지 임베드 플레이어 구현을 사용하여 플레이어, 플러그인 및 CSS의 기능을 테스트합니다 (CSS가 필요한 경우)
  2. 로컬 테스트를 위해 플러그인의 JavaScript와 CSS를 별도의 파일에 넣습니다.
  3. 오류가 발생하면 플러그인 코드와 CSS를 서버에 배포합니다.
  4. Studio를 사용하여 플레이어에 플러그인과 CSS를 추가합니다.
  5. iframe 구현이 더 적합하다고 판단되면 인페이지 임베드 플레이어 구현을 대체합니다 (다음 섹션에서 자세히 설명).

이 단계에 대한 자세한 내용은단계별 을 참조하십시오. 플러그인 개발가이드.

iframe 또는 인페이지 임베드

브라이트코브 플레이어에 대한 향상된 기능을 개발할 때 코드가 iframe 또는 인페이지 임베드 구현에 가장 적합한지 결정해야 합니다. 모범 사례는 iframe 구현과 함께 사용할 플러그인을 빌드하는 것입니다. iframe 플레이어를 사용하면 다음과 같은 이점이 있습니다.

  • 기존 자바스크립트 및/또는 CSS와의 충돌 없음
  • 자동 반응성
  • iframe은 소셜 미디어 앱에서 쉽게 사용할 수 있습니다 (또는 동영상이 다른 앱으로 “여행”해야 할 때마다)

In-Page 임베드 플레이어를 통합하는 것은 더 복잡할 수 있지만, 해당 구현을 중심으로 코드를 계획할 때가 있습니다. 일반화하려면 이 방법은 포함 페이지가 플레이어와 통신해야 할 때 가장 좋습니다. 구체적으로 다음과 같은 몇 가지 예가 있습니다.

  • 포함 페이지의 코드는 플레이어 이벤트를 수신하고 행동해야 합니다.
  • 플레이어는 포함 페이지의 스타일을 사용합니다.
  • iframe은 포함 페이지에서 리디렉션하는 것처럼 앱 로직이 실패합니다.

최종 구현에서 iframe 코드를 사용하지 않더라도 JavaScript용 플러그인과 CSS용 별도의 파일과 함께 인페이지 포함 코드를 사용할 수 있습니다. 이렇게 하면 로직을 캡슐화하여 여러 플레이어에서 쉽게 사용할 수 있습니다.

구현 선택

여기에 설명 된 플레이어는 iframe 삽입에서 작동하지만 모서리를 둥글게하고 컨트롤을 계속 표시하기 위해 플레이어에 일부 CSS가 적용됩니다. 이 스타일은 ID가있는 플레이어에 따라 다르며 인 페이지 삽입에서만 작동합니다. 인 페이지 삽입이 여기에 사용됩니다.

플레이어 만들기

플레이어를 만들고 비디오를 할당하려면 다음 단계를 따르십시오.

  1. 플레이어모듈에서 새 플레이어를 생성하거나 기존 플레이어를 사용합니다.
  2. 플레이어 설정을 편집하여 크기를 변경하십시오.
    • 고정 크기 선택
    • 사용자 정의 종횡비 선택
    • 너비를 원하는대로 설정하되신장 ...에 80픽셀
  3. 에서미디어모듈, 선택오디오 전용 항목이 플레이어와 함께 게시합니다.플레이어에 오디오 전용 콘텐츠를 할당하고 게시합니다.
  4. 인페이지 포함 코드를 복사하여 새 HTML 파일에 붙여 넣습니다.
  5. 추가id audioOnly플레이어에게.

맞춤 CSS 추가

이제 페이지 헤드 섹션의 스타일 블록에 사용자 정의 CSS를 추가합니다. 스타일은 세 가지 작업을 수행합니다.

  • 큰 재생 버튼의 배경 제거(플레이어 키에 비해 너무 크기 때문에)
  • 사용자가 마우스를 끄더라도 재생 중에 컨트롤 막대를 계속 표시합니다.
  • 플레이어의 모서리를 둥글게하여 좀 더 오디오 플레이어처럼 보이게합니다.

추가해야하는 CSS 코드는 다음과 같습니다.

/* keep the controls visible during playback */
#audioOnly .vjs-control-bar {
    transform: none;
    background-color: transparent;
}
/* round the corners of the player */
#audioOnly {
    border-radius: 1em;
    width: 480px;
    height: 80px;
}
/* remove the background from the big play button */
.video-js.vjs-mouse .vjs-big-play-button {
    background: none;
}
/* round the corners of the dock */
#audioOnly .vjs-dock-text {
    border-radius: 1em 1em 0 0;
}
