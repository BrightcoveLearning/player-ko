플레이어 예제

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 이 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

플레이어를 보기 안팎으로 스크롤합니다. 플레이어가보기로 완전히 스크롤되면 동영상 또는 광고가 재생되기 시작합니다. 플레이어를 화면 밖으로 스크롤하면 동영상 또는 광고 재생이 중지됩니다. 이 예에는 프리 롤, 건너 뛸 수있는 5 초 미드 롤 및 포스트 롤이 있습니다.

참고: 코드 샘플 Brightcove 플레이어 샘플 : 플레이어를 보기로 스크롤 IMA3 광고를 사용하지 않는다는 점을 제외하면 이 샘플과 매우 유사합니다.

이 응용 프로그램의 기본 로직은 다음과 같습니다.

플러그인 코드

일반적으로 JavaScript를 브라이트코브 플레이어 플러그인으로 변환할 때는 명목상 변경이 한 가지 필수 변경 사항은ready()메서드의 표준 사용을 플러그인을 정의하는 코드로 대체하는 것입니다.

다음은 플레이어와 함께 작동하는 JavaScript 코드에 매우 일반적으로 사용되는 시작입니다.

videojs.getPlayer('myPlayerID').ready(function() {
  var myPlayer = this;
  ...
});

표준 구문을 사용하여 브라이트코브 플레이어 플러그인을 시작하도록 첫 번째 줄을 변경합니다.

videojs.registerPlugin('pluginName', function(options) {
  var myPlayer = this;
  ...
});

앞서 언급했듯이이 문서의 해당 GitHub 저장소에서 플러그인의 JavaScript 코드를 볼 수 있습니다. scroll-into-view-ads.js .
