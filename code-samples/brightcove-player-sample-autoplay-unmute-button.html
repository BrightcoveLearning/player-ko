2018년 4월부터 Chrome은 음소거 해제된 동영상의 자동 재생을 중단했습니다. Chrome이 음소거되지 않은 동영상을 자동 재생하는 특별한 상황이 있습니다. Google의 보기자동재생 정책 변경자세한 내용은 문서를 참조하십시오.

플레이어 예제

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 이 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

다음 비디오는 비디오를 실행하는 브라우저 / 플랫폼에 따라 다르게 작동합니다. 비디오가 오디오와 함께 자동 재생되면 재생됩니다. 그렇지 않은 경우 비디오 재생이 시작되지만 음소거 해제 버튼이 나타납니다. 이 버튼을 클릭하면 비디오의 오디오가 들립니다. 일반적으로 iOS, Safari 또는 Chrome을 사용할 때 동영상이 음소거되지 않은 경우 동영상이 자동 재생되지 않습니다. 이 샘플은 동영상을 자동 재생하는 방법을 보여주고 음소거 해제를위한 명확한 버튼을 제공합니다.

IMA3 프리 롤을 사용하는 경우프리 롤 광고 사용이 문서의 섹션. 이 섹션에서는 IMA 프리 롤 광고에서 기능이 작동하도록하기 위해 다음 CodePen에 표시된 코드에 적용해야하는 사소한 변경 사항에 대해 자세히 설명합니다.

펜보기음소거 해제 버튼으로 자동 재생게시자 : Brightcove Learning Services ( 뿡 빵뀨 ) 의 위에 CodePen .

개발 시퀀스

권장되는 개발 시퀀스는 다음과 같습니다.

  1. 인페이지 임베드 플레이어 구현을 사용하여 플레이어, 플러그인 및 CSS의 기능을 테스트합니다 (CSS가 필요한 경우)
  2. 로컬 테스트를 위해 플러그인의 JavaScript와 CSS를 별도의 파일에 넣습니다.
  3. 오류가 발생하면 플러그인 코드와 CSS를 서버에 배포합니다.
  4. Studio를 사용하여 플레이어에 플러그인과 CSS를 추가합니다.
  5. iframe 구현이 더 적합하다고 판단되면 인페이지 임베드 플레이어 구현을 대체합니다 (다음 섹션에서 자세히 설명).

iframe 또는 인페이지 임베드

브라이트코브 플레이어에 대한 향상된 기능을 개발할 때 코드가 iframe 또는 인페이지 임베드 구현에 가장 적합한지 결정해야 합니다. 모범 사례는 iframe 구현과 함께 사용할 플러그인을 빌드하는 것입니다. iframe 플레이어를 사용하면 다음과 같은 이점이 있습니다.

  • 기존 자바스크립트 및/또는 CSS와의 충돌 없음
  • 자동 반응성
  • iframe은 소셜 미디어 앱에서 쉽게 사용할 수 있습니다 (또는 동영상이 다른 앱으로 “여행”해야 할 때마다)

In-Page 임베드 플레이어를 통합하는 것은 더 복잡할 수 있지만, 해당 구현을 중심으로 코드를 계획할 때가 있습니다. 일반화하려면 이 방법은 포함 페이지가 플레이어와 통신해야 할 때 가장 좋습니다. 구체적으로 다음과 같은 몇 가지 예가 있습니다.

  • 포함 페이지의 코드는 플레이어 이벤트를 수신하고 행동해야 합니다.
  • 플레이어는 포함 페이지의 스타일을 사용합니다.
  • iframe은 포함 페이지에서 리디렉션하는 것처럼 앱 로직이 실패합니다.

최종 구현에서 iframe 코드를 사용하지 않더라도 JavaScript용 플러그인과 CSS용 별도의 파일과 함께 인페이지 포함 코드를 사용할 수 있습니다. 이렇게 하면 로직을 캡슐화하여 여러 플레이어에서 쉽게 사용할 수 있습니다.

애플리케이션 흐름

이 응용 프로그램의 기본 로직은 다음과 같습니다.

  • 사용play()비디오 재생을 시도하는 방법입니다. 메서드에 대한 호출은 JavaScript에서 반환 된 값을 보유 할 변수에 할당됩니다. 약속 .
  • 약속이 해결되면 (비디오 재생 중) 플레이어 음소거를 해제하고 볼륨 수준을 설정합니다.
  • Promise가 거부되면 (비디오가 재생되지 않음) 다음을 수행하십시오.
    • 플레이어를 음소거하고 비디오 재생을 시작합니다 (플레이어가 음소거되었으므로 이제 수행됩니다).
    • HTML 버튼 요소를 동적으로 생성하여음소거 해제단추.
    • 추가하다 click버튼의 이벤트 리스너.
    • 버튼의 텍스트, 스타일 등을 구성합니다.
    • 플레이어 위에 버튼을 놓습니다.

프리 롤 광고 사용

음소거 해제 버튼이 표시되면 광고가 재생되므로 콘텐츠 플레이어가 아닌 광고 플레이어의 음소거를 해제해야합니다. 이렇게하려면 클릭 핸들러에 다음을 사용합니다.

button.addEventListener("click", function() {
  myPlayer.ima3.adPlayer.muted(false);
  myPlayer.ima3.adPlayer.volume(volumeLevel);
  //myPlayer.muted(false);
  //myPlayer.volume(volumeLevel);
  playerContainer.removeChild(button);
}

콘텐츠 플레이어 볼륨 제어를위한 두 줄의 코드는 주석 처리되고 비교를 위해서만 남아 있습니다.
