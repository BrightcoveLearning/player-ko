출시 전 플레이어가 필요한 이유는 무엇입니까?

시험판을 포함하여 최신 버전의 Brightcove Player에 대해 자주 테스트할 수 있습니다. 이 문서는 새로운 시험판을 사용할 수 있을 때마다 업데이트될 플레이어를 구성하는 방법을 보여줍니다. 설정하여"version_alias": "prerelease"구성 옵션을 사용하면 이 플레이어가 항상 최신 시험판 버전을 갖게 될 것입니다.

스튜디오에서선수항상 최신 시험판에서 사용할 플레이어를 설정할 수 있는 모듈:

작동 방식

현재 주 버전에 대한 새 시험판 버전이 릴리스될 때마다 Player Management API는 마스터 분기에 시험판 버전 별칭이 있는 모든 플레이어를 다시 게시합니다.

API로 시험판 플레이어 만들기

다음 curl 문은 항상 최신 시험판을 사용하도록 Player Management API를 사용하는 플레이어를 만듭니다.

  curl \
    --header "Content-Type: application/json" \
    --user $EMAIL \
    --request POST \
    --data '{
        "name": "Player",
        "configuration": {
          "player": {
            "template": {
              "name": "single-video-template",
              "version_alias": "prerelease"
            }
          }
        }
      }' \
      https://players.api.brightcove.com/v1/accounts/$ACCOUNT_ID/players
