브라이트코브는 모든 브라이트코브 서비스 및 애플리케이션의 상태를 지속적으로 모니터링합니다. 서비스가 중단되면 시스템 상태 페이지에 메모가 게시됩니다. 예정된 모든 유지 관리 통지도 이 페이지에 게시됩니다.

시스템 상태 페이지에 액세스하려면 https://status.brightcove.com 를방문하거나 탐색 헤더에서 지원 아이콘 () 을 누른 다음 시스템 상태를선택합니다.

게시된 모든 알림에는 다음 정보가 포함됩니다.

  • 사고 상태
    • 운영 (녹색)
    • 알림 (파란색 - 시스템이 현재 작동 중이지만 지연 또는 간헐적인 중단의 영향을 받음)
    • 부분 서비스 중단 (노란색)
    • 전체 서비스 중단 (빨간색)
  • 구성 요소 - 서비스 또는 영향을 받는 지역
  • 위치
    • APAC
    • 아메리카
    • EMEA
    • 일본
  • 현재 상태 및 상태 주에는 다음이 포함됩니다.
    • 조사 - 엔지니어가 문제를 조사하고 있습니다.
    • 식별됨 - 엔지니어가 문제를 식별하여 수정하고자 합니다.
    • 모니터링 - 문제 해결 및 모든 것이 정상인지 확인하기 위한 엔지니어링 모니터링
    • 해결됨 - 문제 해결됨

게시된 모든 메시지는 영어로 표시되며 사용 시간은 동부 표준시 (GMT-5) 입니다.

시스템 상태 기록 보기

시스템 상태 기록을 보려면 페이지 하단의기록링크를 클릭합니다.

시스템 상태 변경 구독

시스템 상태가 변경될 때 알림을 받도록 가입할 수 있습니다. 이 섹션은 다음과 같이 구성됩니다.

  • 이메일 구독
  • 웹훅 구독하기
  • 구독 관리

이메일 구독

이메일로 알림을 받으려면구독버튼을 클릭하고 이메일 주소를 입력한 다음구독을 클릭합니다 .

웹훅 구독하기

웹훅은 HTTP POST를 사용하여 정의된 콜백입니다. 상태 업데이트가 있을 때 콜백에서 JSON 객체를 게시할 URL을 지정합니다.

시스템 상태가 변경될 때 알림을 받도록 등록하려면 SUBSCRY버튼을 선택합니다. 그런 다음 이메일 주소와 함께 JSON 페이로드를 수신할 URL을 입력합니다.

시스템 상태 변경이 전송되면 웹훅 주소는 다음과 유사한 JSON 객체를 수신해야 합니다.

구독 관리

구독을 마치면 구독을 관리할 수 있는 구독 관리 페이지로 연결되는 링크가 포함된 이메일을 받게 됩니다. 알림 수신에 관심이 있는 구성 요소/제품/지역을 선택할 수 있습니다. 선택한 다음 구독저장을 클릭합니다 .

구독을 취소하려면구독 관리 페이지 하단의 구독 취소링크를 클릭합니다.

Slack에서 알림 받기

Slack에는 상태 업데이트를 연결하는 데 사용할 수 있는 자체 웹훅 시스템이 있습니다. 자세한 내용은 Slack 설명서를 참조하십시오.
