플레이어 개발자 기본 사항: 고급 플레이어 코드

이 항목에서는 프로그래밍 방식으로 비디오를 자동 재생하는 방법을 알아 봅니다.
 

단계

  1. 프로그래밍 방식으로 제어하기위한 표준 모범 사례 코드로 시작 플레이어. 
    <video id="myPlayerID"
  data-video-id="5785633544001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;

  });
</script>
  2. 익명 기능에서videojs.getPlayer()메서드를 사용하여 변수 할당에 따라 플레이어를 음소거하여 자동 재생되도록합니다. 
    myPlayer.muted(true);
  3. 플레이어의on()이벤트 처리기 함수를 추가하는 메서드loadedmetadata행사. 
    myPlayer.on('loadedmetadata', function(){

});
  4. 이벤트 핸들러에서 동영상을 재생합니다. 
    myPlayer.play()
