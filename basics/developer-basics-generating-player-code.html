플레이어 개발자 기본 사항: 플레이어 코드 생성

이 항목에서는 나중에 플레이어 개발에 사용할 수있는 플레이어 코드를 생성하는 방법을 배웁니다.
  1. Studio에서선수기준 치수.
  2. [ 새 플레이어]버튼을 클릭합니다.
  3. 플레이어와 이름을 지정한 다음저장 .
  4. MEDIA모듈을 엽니다.
  5. 게시 할 동영상을 클릭하세요.
  6. 플레이어 속성 페이지에서게시 및 삽입버튼, 드롭 다운에서 선택웹 플레이어 .
  7. 실제 플레이어의 경우 방금 만든 플레이어를 선택합니다.
  8. 에서코드 복사섹션 선택많은이행.
  9. 다음을 설정하십시오.
    • 크기 조정 : 결정된
    • 가로 세로 비율 : 비디오 일치
    • 플레이어 폭 : 원하는 너비
  10. 마지막으로 코드를 복사합니다.
