플레이어 개발자 기본 사항: 커스텀 플러그인-Studio 사용

이 항목에서는 Studio를 사용하여 Brightcove Player에 사용자 지정 플러그인을 할당하는 방법을 알아 봅니다.
 

단계

  1. 스튜디오에서플레이어모듈을 선택합니다.
  2. 사용자 지정 플러그인을 추가하려는 플레이어를 클릭합니다.
  3. 플레이어 속성의 왼쪽에서플러그인옵션.
  4. 플러그인추가드롭다운에서사용자 지정 플러그인을 선택합니다 .
  5. 다음을 추가하십시오.
    • 플러그인 이름
    • 자바스크립트 URL
    • CSS URL-필요한 경우
    • 옵션 (JSON)-필요한 경우
  6. 플레이어를 저장하십시오.
  7. 게시플레이어.
