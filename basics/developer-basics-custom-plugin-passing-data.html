플레이어 개발자 기본 사항: 사용자 정의 플러그인-데이터 전달

이 주제에서는 사용자 정의 플러그인에 데이터를 전달하는 방법을 학습합니다.
 

단계

  1. 사용자 정의 플러그인 코드에서options플러그인을 정의하는 익명 함수의 매개 변수. 
    videojs.registerPlugin('navigateOnVideoEnd', function (options) {
  var myPlayer = this;
  ...
});
  2. 사용자 정의를 위해 플러그인 코드에서 매개 변수 개체의 원하는 속성을 사용합니다. 
    videojs.registerPlugin('navigateOnVideoEnd', function (options) {
  var myPlayer = this;
  ...
  window.location.href = options.redirectURL;
});
  3. 플러그인을 호출하는 HTML 페이지에서 필수 속성 / 속성을 사용하여 개체를 빌드합니다. 
    <script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this,
       options = {"redirectURL": "http://support.brightcove.com"};
    ...
  });
</script>
  4. 사용자 정의 플러그인을 호출 할 때options매개 변수로 개체 : 
    <script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this,
       options = {"redirectURL": "http://support.brightcove.com"};
    myPlayer.navigateOnVideoEnd(options);
  });
</script>
  5. Studio를 사용하는 경우 옵션 개체를옵션 (JSON)양식 요소 :

전체 코드

플러그인 코드

videojs.registerPlugin('navigateOnVideoEnd', function (options) {
  var myPlayer = this;
  myPlayer.on('ended', function () {
    window.location.href = options.redirectURL;
  });
});
