Brightcove Player와 함께 라이브 SSAI용 플레이어 향상 기능 사용

{{ page.description }}

소개

라이브 SSAI를 위한 Brightcove의 플레이어 향상 기능을 사용하면 광고 수, 광고 카운트다운 타이머, 클릭연결 광고 및 컴패니언 광고를 사용하여 라이브 스트림의 광고 시간을 개선할 수 있습니다. 이러한 라이브 스트림은 Brightcove Player를 사용하여 재생할 수 있습니다.

이 기능을 처음 사용하는 경우 다음을 참조하십시오.

  • 개요: 라이브 SSAI를 위한 플레이어 개선 사항
  • 라이브 SSAI를 위한 플레이어 개선 사항 구현

단계

Brightcove Player를 사용하여 향상된 플레이어 기능으로 라이브 스트림을 재생하려면 다음 단계를 따르십시오.

  1. Video Cloud Studio에서 기존 플레이어를 만들거나 선택합니다. 플레이어기준 치수.

  2. Brightcove Player는 SSAI 플러그인으로 구성해야합니다. 따라 SSAI 플러그인 추가에 설명 된 단계프로그래밍 방식으로 SSAI 구현 SSAI 플러그인 문서의 섹션.

    Live SSAI를 통한 플레이어 향상의 경우 Brightcove Player에는 SSAI 플러그인( videojs-ssai ). 현재 Video Cloud Studio를 사용하여 SSAI 플러그인을 플레이어에 추가할 수 없습니다. 광고 구성 ID가 필요하기 때문입니다.
  3. 워크 플로를 따르십시오. Live SSAI를위한 플레이어 향상 구현문서, 재생 토큰 생성을 포함합니다.
  4. 구현 선택 :
    • 표준 임베드 코드
    • 고급 임베드 코드
    • 라이브 SSAI 소스
  5. 의 워크플로를 계속 진행합니다. 라이브 SSAI를 위한 플레이어 개선 사항 구현문서.

표준 임베드 코드

Video Cloud Studio에서 비디오를 게시 할 때미디어모듈에서 표준 소스 코드를 사용하여 라이브 스트림을 재생할 수 있습니다. 다음 단계를 따르십시오.

  1. 미디어모듈에서 라이브 스트리밍을 위해 원격 비디오를 선택하고 SSAI 지원 플레이어로 게시합니다.
  2. 표준 소스 코드를 복사하여 웹 페이지에 붙여 넣습니다.

  3. 에서<iframe>태그, 당신은src속성. 이 끝에adConfigId값이있는 URL 매개 변수재생 토큰 .

    <iframe src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/
index.html?videoId=6089818309001&adConfigId=live.aaB61234aabbccdd567890"
  allowfullscreen=""
  allow="encrypted-media"
  width="960" height="540">
</iframe>
  4. 플레이어가 실시간 스트림을 재생할 준비가 되었습니다.

고급 임베드 코드

Video Cloud Studio에서 비디오를 게시 할 때미디어모듈에서 고급 소스 코드를 사용하여 라이브 스트림을 재생할 수 있습니다. 다음 단계를 따르십시오.

  1. 미디어모듈에서 라이브 스트리밍을 위해 원격 비디오를 선택하고 SSAI 지원 플레이어로 게시합니다.
  2. 고급 포함 코드를 복사하여 웹 페이지에 붙여 넣습니다.

  3. 에서<video-js>태그, 추가data-ad-config-id당신의 가치를 가진 속성재생 토큰 .

    <video-js
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id="6089818309001"
  data-ad-config-id="live.aaB61234aabbccdd567890"
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
  4. 플레이어가 실시간 스트림을 재생할 준비가 되었습니다.

라이브 SSAI 소스

VOD SSAI 소스와 달리 Live SSAI 소스에는 초기 VMAP가 포함되지 않습니다. 라이브 SSAI 스트림은 광고 시간이 언제 포함 될지 미리 알지 못하기 때문입니다. 대신 광고 시간은 라이브 작업에 추가하는 큐 포인트에서 트리거됩니다.

Brightcove Live SSAI 스트림은 특정 명명 체계를 사용하는 표준 HLS 매니페스트 URL로 시작합니다.

https://.../playlist_ssaiM.m3u8

SSAI 플러그인은_ssaiM.m3u8접미사 및 일반 HLS MIME 유형

Video Cloud Studio에서 비디오를 게시 할 때미디어모듈에서 다음과 같이 라이브 SSAI 스트림을 재생할 수 있습니다.

  1. 미디어모듈에서 라이브 스트리밍을 위해 원격 비디오를 선택하고 SSAI 지원 플레이어로 게시합니다.
  2. 고급 포함 코드를 복사하여 웹 페이지에 붙여 넣습니다.

  3. 페이지 코드에 다음 코드를 추가하십시오.

    <video-js id="myPlayerID"
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id=""
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.src({
      type: 'application/x-mpegURL',
      src: 'https://path/to/playlist_ssaiM.m3u8'
    })
  });
</script>
  4. 플레이어가 실시간 스트림을 재생할 준비가 되었습니다.
