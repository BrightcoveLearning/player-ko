플레이어 모듈은 Google의 유니버설 애널리틱스 데이터 모델을 지원합니다. 현재 최신 Google Analytics 4 업데이트는 지원되지 않습니다. Brightcove는 플레이어 모듈 및 Brightcove 플레이어와 통합할 때 Universal Analytics를 사용할 것을 권장합니다.

소개

구성 프로세스는 다음 작업으로 구성됩니다.

  • Brightcove Player에서 지원하는 이벤트 중에서 추적하려는 이벤트를 선택하십시오.
  • Google 애널리틱스 플러그인에서 지원하는 측정 기준에서 사용하려는 맞춤 측정 기준을 선택하세요.
  • Google 애널리틱스에서 선택한 맞춤 측정 기준을 만듭니다.
  • Video Cloud Studio에서 Google Analytics 플러그인에 대한 구성 정보를 추가합니다. 플레이어의 JSON 편집기를 사용하여이 작업을 수행합니다.

이벤트 선택

JSON 구성에서 추적하려는 이벤트를 나열합니다. 다음 중 추적 할 항목을 결정하십시오.

  • 플레이어_로드
  • 비디오_인게이지먼트
  • 비디오_임프레션
  • 플레이_요청
  • ad_start
  • ad_end
  • 오류
  • 비디오_뷰
  • 비디오_완성

맞춤 측정 기준 선택

Google 애널리틱스 데이터에서 사용할 수있는 8 개의 맞춤 측정 기준이 있습니다. 다음 중 사용할 항목을 결정하십시오.

  • bcvideo_account
  • bcvideo_platform_version
  • bcvideo_player
  • bcvideo_range
  • bcvideo_session
  • bcvideo_tags
  • bcvideo_video_duration
  • bcvideo_video_seconds_viewed

맞춤 측정 기준 만들기

사용중인 각 맞춤 측정 기준을 만들어야합니다. 이 문서에있는 Google의 단계는 프로세스를 안내합니다. 맞춤 측정 기준 및 측정 항목 생성 및 수정 .

각 맞춤 측정 기준을 생성하는 동안범위 . 이러한 맞춤 측정기준의 경우 값을 사용합니다. 때리다. 또한 각 맞춤 측정 기준을 만들 때 다음 스크린 샷에 표시된 것처럼 색인 또는 시퀀스와 번호가 연결됩니다.

구성 JSON에 색인 번호가 필요하므로 생성 된 맞춤 측정 기준 표를 사용할 수 있어야합니다.

차원을 생성 할 때 더 친숙한 이름을 선택할 수 있습니다. 예를 들어 다음을 고려하십시오.

  • 에 대한 맞춤 측정기준을 만들 때 bcvideo_video_duration당신은 단순히 이름을지속보고서에 더 친숙한 이름을 사용하기를 원하기 때문입니다.
  • 생성 시 다음과 연결된 인덱스지속 ~이다 7 .
  • 구성 JSON에서 다음 할당을 사용합니다. 
    "bcvideo_video_duration": "dimension7"
    색인 번호는 플러그인의 내부 이름과 친숙한 이름을 연결합니다.

플레이어에 JSON 구성 추가

사용할 이벤트와 사용할 사용자 지정 차원에 대한 결정을 내렸고 후자를 만들었습니다. 플레이어 구성에 포함 할 JSON을 빌드 할 준비가되었습니다.

코드의 각 논리적 그룹을 설명하는 표가있는 예가 아래에 나와 있습니다. 이 가이드에 따라 고유 한 JSON을 빌드 할 수 있습니다.

4.18 JSON 경계옵션플러그인에 대한 정의
5.11 사용자 정의 정의에 대한 JSON 경계

구성 JSON이 완료되면 다음 단계에 따라 Brightcove Player의 구성에 붙여 넣습니다.

  1. 비디오 클라우드 스튜디오에 로그인합니다.
  2. 다음으로 이동선수모듈을 클릭하고 새 구성을 추가 할 플레이어를 클릭합니다.
  3. 왼쪽 탐색에서 JSON 편집기 .
  4. 이미 플레이어에 대해 구성된 플러그인이있는 경우 해당 섹션에 코드를 붙여 넣을 수 있습니다. 그렇지 않으면 플러그인 용 JSON을 추가해야합니다. 두 경우 모두 결과는 다음과 유사해야하지만 물론 플레이어 정보와 JSON을 사용합니다 (플러그인의 JSON 구성은 노란색으로 강조 표시됨). 
    { "compatibility": true, "video_cloud": { "policy_key": "BCpkADawqM2w1K7269Ic17hjK0O2-ibj221pVcELpF1l5nZuiL-123456789123456789" } , "플레이어": { "template": { "name": "single-video-template", "version": "6.48.5" } }, "플러그인": [ { "registry_id": "@brightcove/videojs-bc-ga", "version": "1.x", "options": { "paramsToTrack": { "bcvideo_video_seconds_viewed": "dimension1", "bcvideo_range": "dimension2", "bcvideo_video_duration": "dimension3", "bcvideo_player": "dimension4", "bcvideo_session": "dimension5" } , "eventsToTrack": { "video_impression": "Video Impression", "play_request": "Play Request", "video_engagement": "Video Engagement" } , "추적자": "UA-123456-1" } } ] }
  5. 변경 사항을 저장하고 플레이어를 게시하면 Google Analytics 플러그인을 사용할 수 있습니다.
