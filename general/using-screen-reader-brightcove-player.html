Brightcove Player와 함께 화면 판독기 사용

{{ page.description }}

스크린 리더를 사용하면 시각 장애가있는 사용자가 비디오 내의 작업을들을 수 있습니다. 정의 된 WebVTT 설명 파일을 읽어이를 수행합니다. 이것은 비디오에서 대화가 제한적이거나 전혀없는 경우에 유용합니다.

이 예에서는 Mac OS X 운영 체제와 통합 된 Apple의 VoiceOver 앱을 사용합니다. 자세한 내용은 VoiceOver 시작하기문서.

Brightcove Player와 호환되는 화면 판독기 목록은 다음을 참조하십시오. Brightcove 플레이어 접근성문서.

이 예는 Chrome 웹 브라우저에서 가장 잘 작동합니다. Safari 브라우저는 현재 설명 텍스트 트랙을 지원하지 않습니다. VoiceOver에 대한 기본 지원 만있는 Firefox와 같은 다른 브라우저에서 문제를 찾을 수 있습니다.

리소스

몇 가지 유용한 리소스는 다음과 같습니다.

  • 텍스트 트랙 작업을위한 Video.js 가이드 .
  • HTML5 비디오 캡션 메이커

 

플레이어

이 예제는 WebVTT 파일을 사용하여 비디오에서 일어나는 일을 설명합니다. 화면 판독기가 설명 텍스트를 읽으므로 시각 장애가있는 사용자가 비디오의 작업을 따를 수 있습니다.

 

이 비디오에 스크린 리더 사용

아래 단계에 따라 설명 및 스크린 리더를 켭니다.

  1. 위의 동영상 재생을 시작하고 플레이어 컨트롤이 표시되면 중지합니다.
  2. 컨트롤 막대에서 마우스를설명아이콘을 클릭하고 영어 설명을 선택합니다. ko 설명 .
  3. 이제 플레이어에 표시된 설명 파일의 텍스트를 볼 수 있습니다.
  4. 이제 MacOS 컴퓨터와 함께 제공되는 VoiceOver 스크린 리더를 활성화합니다. 고르다 CMD + F5 VoiceOver를 켜려면
  5. VoiceOver 모달 대화 상자가 나타나면 VoiceOver 사용단추.
  6. VoiceOver 제어 막대가 나타나 비디오에 대한 설명 파일의 텍스트를보고들을 수 있습니다. 또한 컴퓨터에서 수행하는 모든 작업을보고들을 수 있습니다.
  7. 고르다 CMD + F5 VoiceOver를 끄려면

비디오 준비

플레이어 컨트롤 막대에서 설명 아이콘을 사용하려면 WebVTT 텍스트 트랙 파일을 생성하여 비디오와 연결해야합니다. 이를 수행하는 방법을 배우려면 다음 단계를 따르십시오.

  1. 비디오의 동작을 설명하는 텍스트 트랙 파일을 만듭니다. 이 파일은 이후 단계에서 설명 유형으로 동영상에 적용한다는 점을 제외하면 캡션 파일을 만드는 것과 동일합니다. 이 파일을 만드는 방법에 대한 자세한 내용은미디어 모듈을 사용하여 비디오에 캡션 추가문서.

    텍스트 트랙 파일은 다음과 유사해야합니다.

    WEBVTT

00:00:00.000 --> 00:00:05.000
[Herons...]

00:00:05.000 --> 00:00:10.000
[Great Blue ones...]

00:00:10.000 --> 00:00:15.000
[Nesting...]

00:00:15.000 --> 00:00:20.000
[Enjoy...]

00:00:20.000 --> 00:00:25.000
[Florida 2016]

  2. Brightcove Player 고객은 다음 방법을 배울 수 있습니다. 비디오에 캡션 추가프로그래밍 방식으로.

    다음으로 텍스트 트랙 파일을 비디오와 연결합니다. Video Cloud 고객은 Studio에 로그인하여미디어기준 치수.

  1. 찾기텍스트 트랙섹션을 선택하고편집하다단추.
  2. 텍스트 트랙 파일을 업로드하거나 외부 URL을 지정하십시오.
  3. 들어가다두 글자 언어 코드텍스트 트랙 데이터의 언어입니다. 이 예에서는 다음을 입력합니다. ko영어. 화면 판독기가 지원하는 언어를 확인하십시오.
  4. 설정종류텍스트 트랙의설명 .
  5. 기본값 유지 MIME 유형값. 선택할 수 있습니다기본이 언어를 기본 언어로 설정합니다.
  6. 선택텍스트 트랙 추가버튼을 눌러 텍스트 트랙 정보를 저장합니다.
  1. 이제 Brightcove Player 및 스크린 리더에서이 비디오를 사용할 준비가되었습니다.
