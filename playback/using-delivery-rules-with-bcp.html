소개

Brightcove의 전달 규칙을 사용하면 적시 매니페스트 생성 기능을 활용하여 사용자 지정 규칙을 사용하여 콘텐츠가 뷰어에게 전달되는 방식을 제어할 수 있습니다.

배달 규칙에 대한 자세한 내용은 다음을 참조하십시오.

  • 개요: 배달 규칙
  • 배달 규칙 구현
  • 네이티브 SDK와 함께 배달 규칙 사용

브라이트코브 플레이어 구성

다음과 같은 방법으로 Brightcove Player에서 제공 규칙을 구성 할 수 있습니다.

플레이어 관리 API

Player Management API로 플레이어를 구성 할 때 다음을 설정할 수 있습니다. delivery_config_id다음과 같이 구성 속성 :

{
  "delivery_config_id": "abcdef",
  ...
}

표준 임베드 코드

표준 소스 코드 (iframe)를 사용하여deliveryConfigId검색어 매개 변수 :

<iframe src="https://players.brightcove.net/.../index.html?deliveryConfigId=abcdef">

이렇게 하면 플레이어 구성에 설정된 모든 딜리버리 규칙이 재정의됩니다.

고급 임베드 코드

고급 임베드 코드 (인 페이지 임베드)를 사용하여data-delivery-config-id속성:

<video-js data-delivery-config-id="abcdef">

이렇게 하면 플레이어 구성에 설정된 모든 딜리버리 규칙이 재정의됩니다.

카탈로그 API 사용

자바 스크립트를 사용하면 다음과 같이 전달 규칙을 가져올 수 있습니다. deliveryConfigId카탈로그 API 사용 :

var player = videojs.getPlayer('myPlayerId');
player.ready(function () {
  var catalogParams = {
    deliveryConfigId: delivery_rules_id,
    type: 'video',
    id: video_id,
  };

  player.catalog.get(catalogParams, function (error, video) {
    if (error) {
      throw new Error(error);
    }

    player.catalog.load(video);
  });
});
