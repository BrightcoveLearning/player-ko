소개

Brightcove는 해외에서 중국으로 비디오를 전송하기 위한 솔루션을 제공합니다. 솔루션은 두 부분으로 구성됩니다.

  • 중국 내에서 호스팅되는 프록시 서버
  • 연결된 중국 전용 CDN

프록시는 비디오가 아닌 트래픽(Playback API 요청, Analytics 비콘 및 웹 플레이어 자체)을 중국의 시청자와 외부에 있는 Brightcove 인프라로 라우팅합니다.

CDN은 프록시에 연결되어 비디오 및 이미지 콘텐츠를 처리합니다.

요구 사항

Brightcove China Delivery를 지원하려면 다음 요구 사항이 필요합니다.

설정

  • 플레이어 구성 외에도 설정 요구 사항이 있습니다.

    자세한 내용은개요: 중국 배달

브라이트코브 플레이어

  • Brightcove Player 버전 6.44 이상이 필요합니다. 최신 버전 권장

브라이트코브 플레이어 구성

중국 내에서 비디오를 전송하려면 Brightcove China Delivery 프록시를 사용하도록 플레이어를 구성해야합니다.

중국 호환 플레이어를 만들려면 다음 단계를 따르세요.

  1. Video Cloud Studio에서선수기준 치수. 기존 플레이어를 사용하거나 새 플레이어를 만듭니다.

    Brightcove Player 버전 6.44 이상을 사용하십시오(최신 릴리스 권장).
  2. 플레이어의 속성을 열려면 플레이어의 이름 링크를 선택하십시오.

  3. 왼쪽 탐색에서 JSON 편집기를 선택합니다 .

  4. 프록시를 설정하려면 다음을 추가하십시오.

    "china_proxy_domain": "china.example.com"
    호스트 이름만 사용해야 합니다. 프로토콜(예: https://)이 없고 슬래시를 포함한 특수 문자가 없습니다.

  5. 코드는 다음과 유사해야 합니다.

  6. 딸깍 하는 소리구하다 , 플레이어를 게시합니다.
  7. 플레이어를 포함하거나 공유할 때 포함 코드에서 Brightcove Player CDN을 사용합니다. 
    <script
  src="//players.brightcove.net/1752604059001/jnOD8yLME_default/index.min.js">
</script>
  8. Brightcove Player CDN을 변경하여 China Delivery 도메인을 사용하십시오. 
    <script
  src="//china.example.com/1752604059001/jnOD8yLME_default/index.min.js">
</script>
    사용players.brightcove.net도메인으로 인해 중국에서 콘텐츠를 재생할 수 없고 플레이어 로드 시간이 저하됩니다.

노트

와china_proxy_domain설정하면 플레이어가 시작한 모든 요청이 해당 도메인 내의 URL로 전송됩니다. 위의 샘플 값 사용 ( china.example.com ) 사용자는 다음에 대한 요청을 볼 수 있습니다.

체계 통로
플레이어 CDN //china.example.com/players / ...
재생 API //china.example.com/playback/v1 / ...
해석학 //china.example.com/v2/tracker ...

현지화

와china_proxy_domain설정하면 Brightcove Player가자동으로두 가지 언어로 구성됩니다.

  • zh-Hans: 중국어 간체 (기본값)
  • zh-Hant: 중국어 번체

설정china_proxy_domain다음 구성을 설정하는 것과 동일한 효과가 있습니다 (필수 사항은 아님).

"language": "zh-Hans",
"languages": ["zh-Hans", "zh-Hant"],

이 기본 효과china_proxy_domain값이 제공되면 발생하지 않습니다. language또는languages구성! 기본 동작을 재정의하려면 이러한 구성을 혼합하고 일치시킬 수 있습니다.

더 많은 언어 추가

더 많은 언어가zh-Hans과zh-Hant필요한 경우 플레이어에 언어 구성을 추가 할 수 있습니다. 이 구성은 다음과 같습니다.

"china_proxy_domain": "china.example.com",
"languages": ["zh-Hans", "zh-Hant", "en", "jp"],

이 예제의 플레이어는 여전히 기본적으로zh-Hans이지만 영어와 일본어도 추가합니다.

기본 언어 변경

다른 기본 언어를 구성하려면zh-Hans , 추가하다language구성. 구성은 다음과 유사해야합니다.

"china_proxy_domain": "china.example.com",
"language": "zh-Hant",
