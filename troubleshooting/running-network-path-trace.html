네트워크 경로 추적 실행

{{ page.description }}

개요

그만큼traceroute진단 도구는 컴퓨터에서 문제의 웹 사이트까지의 경로를 추적합니다. 이 도구는 네트워크를 통해 일련의 패킷을 보내고 경로의 각 라우터로부터 응답을받습니다. 이를 통해 경로에 문제가 있는지 확인합니다.

지원과 함께 작업 할 때 네트워크 경로를 추적하여 해당 웹 사이트로가는 정보를 제공하라는 요청을받을 수 있습니다.

macOS 또는 Linux에서 추적

네트워크 경로 추적을 실행하려면 다음 단계를 따르십시오.

  1. 터미널 앱의 창을 엽니 다.
  2. 해당 웹 사이트에 다음 명령을 입력합니다. 
    traceroute your_domain.com

    네트워크에서 IPv6 (인터넷 프로토콜 버전 6) 을 사용한다는 것을 알고 있는 경우 다음 명령을 입력합니다.

    traceroute6 your_domain.com
  3. 사이트로 이동한 트래픽 경로가 표시됩니다. 라우터에서 응답이 수신됨에 따라 작성됩니다. 첫 번째 줄은 라우터를 나타냅니다. 다음 줄은 ISP에서 가져온 것이며 패킷이 웹 사이트에 도달할 때까지 계속됩니다. 출력은 다음과 비슷해야 합니다.
  4. 출력을 이해하는 것에 대해 걱정하지 마십시오. 터미널 창에서편집하다 > 모두 선택 . 그런 다음 출력을 클립 보드에 복사합니다.
  5. 출력물을 이메일에 붙여넣고 해당 지원 담당자에게 보냅니다.

Windows에서 추적

네트워크 경로 추적을 실행하려면 다음 단계를 따르십시오.

  1. 열기스타트메뉴 및 선택운영 .
  2. 유형cmd그리고 선택확인 .
  3. 명령 프롬프트가 열립니다. 해당 웹 사이트에 다음 명령을 입력합니다. 
    tracert your_domain.com

    네트워크에서 IPv6 (인터넷 프로토콜 버전 6) 을 사용한다는 것을 알고 있는 경우 다음 명령을 입력합니다.

    tracert -6 your_domain.com
  4. 사이트로 이동한 트래픽 경로가 표시됩니다. 라우터에서 응답이 수신됨에 따라 작성됩니다. 첫 번째 줄은 라우터를 나타냅니다. 다음 줄은 ISP에서 가져온 것이며 패킷이 웹 사이트에 도달할 때까지 계속됩니다.
  5. 출력을 이해하는 것에 대해 걱정하지 마십시오. 출력을 클립 보드에 복사합니다.
  6. 출력물을 이메일에 붙여넣고 해당 지원 담당자에게 보냅니다.
