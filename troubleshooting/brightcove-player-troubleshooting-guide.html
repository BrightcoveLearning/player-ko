지원되는 브라우저

사용 가능한 브라우저 업데이트를 정기적으로 확인하여 최신 및 최고의 수정 사항 및 성능을 확보하는 것이 좋습니다.

지원되는 브라우저의 전체 목록은 Brightcove Player 시스템 요구 사항문서.

일반적인 오류 메시지

다음은 동영상을 재생하려고 할 때 나타날 수있는 오류 메시지 목록과 문제 해결을위한 권장 단계입니다.

미디어_ERR_중단됨

  • 오류 코드: 1
  • 헤드라인: 비디오 다운로드가 취소되었습니다.
  • 메시지: 미디어 재생을 중단했습니다.

가능한 원인

트랙에서 비디오 다운로드 프로세스가 중단되었음을 나타내는 다소 일반적인 브라우저 오류입니다. 다음과 같을 수 있습니다.

  • 네트워크 서비스 중단 또는 충돌과 같은 컴퓨터 문제
  • 브라우저 문제 또는 구성 문제
  • 브라우저가 지원되지 않거나 현재 비디오를 지원하지 않습니다.
  • 브라우저가 동영상을 재생하려고 할 때 방화벽, 웹 프록시 또는 네트워크 문제로 인해 동영상 다운로드가 차단됨

시도해 볼 단계

다음은 문제를 해결하기 위한 몇 가지 단계입니다.

  • 페이지 새로 고침
  • 다른 브라우저를 사용해보십시오

미디어_ERR_네트워크

  • 오류 코드: 2
  • 헤드라인: 비디오 연결이 끊어졌습니다. 인터넷에 연결되어 있는지 확인하십시오.
  • 메시지: 네트워크 오류로 인해 미디어 다운로드가 도중에 실패했습니다.

가능한 원인

가능한 원인은 다음과 같습니다.

  • 비디오 데이터를 차단하는 방화벽
  • 연결을 방해하거나 트래픽을 필터링하는 웹 프록시
  • 필터링 소프트웨어 (예: Sophos 또는 Lightspeed 웹 필터 차단 콘텐츠)
  • 일반적인 네트워크 오류 (예: 홈/학교/개인용 컴퓨터에 대한 인터넷 연결이 실패함)

시도해 볼 단계

다음은 문제를 해결하기 위한 몇 가지 단계입니다.

  • 해당 컴퓨터에서 인터넷이 올바르게 작동하는지 확인합니다.
  • 삭제되었을 수 있는 연결을 다시 시작할 수 있는 페이지 새로 고침
  • 다른 동영상을 사용해 보세요. 그게 효과가 있나요? 그렇지 않은 경우 콘텐츠 전체에 도달할 수 없음을 나타냅니다.

미디어_ERR_디코드

  • 오류 코드: 3
  • 헤드라인: 동영상이 잘못되었거나 브라우저에서 재생할 수 없는 형식입니다.
  • 메시지: 미디어 재생이 손상 문제로 인해 중단되었거나 미디어가 브라우저에서 지원하지 않는 기능을 사용했기 때문에 중단되었습니다.

가능한 원인

브라우저가 요청한 콘텐츠를 재생할 수 없습니다. 이는 재생 중 또는 시작 시점 또는 시작 시점에 발생할 가능성이 더 높습니다.

시도해 볼 단계

다음은 문제를 해결하기 위한 몇 가지 단계입니다.

  • 페이지를 새로 고침하면 첫 번째 시도에서 부분 다운로드로 인해 재생에 실패한 콘텐츠가 다시 다운로드 될 수 있습니다.
  • 브라우저를 최신 버전으로 업데이트하십시오. Chrome과 Firefox는 업데이트를 요청하지만 확인하는 것이 유용합니다.

미디어_ERR_SRC_NOT_지원

  • 오류 코드: 4
  • 헤드라인: 이 비디오는 이 브라우저에서 사용할 수 없거나 지원되지 않습니다.
  • 메시지: Tamil Aunty

가능한 원인

브라우저가 요청 된 콘텐츠를 사용할 수 없기 때문에 재생할 수 없거나이 특정 브라우저에서 재생할 수 없습니다. 재생 중이 아니라 시작되기 전에 재생이 실패한다는 점에서 오류 3과 다릅니다.

시도해 볼 단계

대부분의 경우 이러한 설정은 모두 정확하지만 IT 팀에서 특별히 설정했을 수 있습니다.

플레이어_ERR_타임아웃

  • 에러 코드: -2
  • 헤드라인: 비디오를 다운로드할 수 없습니다.
  • 메시지: 비디오를 다운로드할 수 없습니다.

플레이어가 45 초 동안 콘텐츠 / 버퍼를 재생할 수 없습니다.

가능한 원인

  • 대역폭 / 네트워크 속도 부족
  • 비디오 데이터를 차단하는 방화벽
  • 연결을 방해하거나 트래픽을 필터링하는 웹 프록시
  • 필터링 소프트웨어 (예: Sophos 또는 Lightspeed 웹 필터 차단 콘텐츠)
  • 일반적인 네트워크 오류 (예: 홈/학교/개인용 컴퓨터에 대한 인터넷 연결이 실패함)
  • 때때로 플레이어 특정 버그로 인해 발생할 수 있습니다.

시도해 볼 단계

정보 캡처

다음 정보는 문제 조사 지원에 도움이됩니다.

최종 사용자 정보

시스템 환경에 대한 다음 정보를 수집하십시오.

  • 스크린 샷을 찍거나 환경 세부 정보를 내 보냅니다. http://supportdetails.com/ . 이러한 세부 정보는 다음과 같습니다.
    • 운영 체제
    • 화면 해상도 / 브라우저 크기
    • 버전을 포함한 웹 브라우저
    • IP 주소
  • 스크린 샷을 찍거나 네트워크 속도 결과를 http://beta.speedtest.net/

유용한 정보

브라우저에서 네트워크 로그를 만들고 저장합니다.

아래 지침은 Chrome용이지만 지원되는 모든 브라우저에 동일한 지침이 적용됩니다.
  • Chrome 개발자 도구를 엽니 다. (Windows에서는 F12, Mac에서는 CMD + OPT + I)
  • 선택콘솔탭을 클릭하여 오류가보고되었는지 확인합니다.
  • 선택회로망탭.
  • 마우스 오른쪽 버튼을 클릭하고콘텐츠와 함께 HAR로 저장 .
  • 을 살리다.har추가 검토를 위해 컴퓨터에 파일 (확장자는 IE의 경우 .xml)입니다.

자세한 지침은네트워크 로그 생성문서.

연락처 지원

오류 메시지 단계 또는 플레이어 디버거로 문제를 해결하지 못한 경우 지원 요청을 제출할 수 있습니다.

필수 문제 해결 정보

다음 정보는 문제 해결에 필수적입니다.

  • 영향을받는 콘텐츠의 URL 또는 동영상 이름
  • 문제가 발생 / 시작된 대략적인 시간 프레임 (지속적이지 않은 경우), 결과 / 배포 / 시스템 문제 / 알려진 문제를 연관시키는 데 사용할 수 있습니다.
  • 문제 유형을 명확히합니다 (재생 안 함,로드 안 함, 일시 중지 및 버퍼링, 품질 불량, 비디오 점프 또는 자르기, 소리 없음 등 ...)
  • 즉시 명확하지 않은 경우 문제를 재현하기 위해 따라야하는 단계를 설명하는 간단한 설명
  • 문제가 발생하는 구성과 이것이 데스크톱 또는 모바일 (OS 플랫폼, 브라우저 및 버전)에 있는지 여부를 나타냅니다.
  • 가능한 한 여러 / 모든 사용자에게 이런 일이 발생하고 하나 또는 여러 네트워크가 영향을받는 것으로 보이는지 확인하십시오.
  • 해당하는 경우 스크린 샷 또는 오류 메시지 텍스트를 제공합니다.
  • 브라우저 콘솔 HAR 로그 (이전 섹션의 안내 참조)
  • 오류 메시지에 표시되는 세션 ID입니다. 세션 ID를 제공하거나 더 쉽게 오류 메시지의 스크린 샷을 찍습니다.
